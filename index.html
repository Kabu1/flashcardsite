Lets learn! Understanding Javascript terms to know where to use them in our code.

Javascript

an object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

Operators perform some operation on single or multiple operands (data value) and produce a resul

Variables

A JavaScript variable is simply a name of storage location. There are two types of variables in JavaScript : local variable and global variable

Functions

A function is a JavaScript procedure is a set of statements that performs a task or calculates a value. To use a function, you must define it somewhere in the scope from which you wish to call it.

Strings

A JavaScript string stores a series of characters like "John Doe". A string can be any text inside double or single quotes: var carName1 = "Volvo XC40"; var carName2 = 'Volvo XC40'; String indexes are zero-based: The first character is in position 0, the second in 1, and so on.

Methods

JavaScript methods are actions that can be performed on objects. A JavaScript method is a property containing a function definition.
